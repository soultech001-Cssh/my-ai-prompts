## 使用说明

1.  **文件获取**：直接使用本目录下的 `.md` 提示词文件。
2.  **运行环境**：建议在 **Gemini** 或其他支持长文本解析及 Markdown 格式的 AI 模型中运行，以获得最佳的结构理解与生成效果。
3.  **自定义**：你可以根据提示词中的 `subject` 字段自由更换人物特征，或调整 `mood` 改变画面氛围。
